YAML ↔ JSON

Convert YAML to JSON and vice versa.

Use Tab and Shift+Tab for indenting .
To move to the YAML editor, press F1.
To move to the JSON editor, press F2.

YAML version:
Kubernetes uses YAML 1.1.

Data doesn't leave your browser.
